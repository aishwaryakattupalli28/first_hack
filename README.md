# Restaurant Pricing Intelligence Dashboard

A smart pricing analytics dashboard that helps restaurants optimize pricing using competitive comparison and revenue simulation.

## Features
- Multi-dish comparison
- Multi-city price adjustment
- Revenue optimization engine
- KPI dashboard
- Historical trend visualization
- Revenue gain distribution

## Tech Stack
- HTML
- CSS
- JavaScript
- Chart.js
